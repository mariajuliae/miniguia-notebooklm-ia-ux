# 📘 Caderno Temático: IA como ferramenta de aprendizagem aplicada à UX em produtos digitais

---

## 🎯 Contexto e Objetivos

Este projeto tem como objetivo explorar o uso da Inteligência Artificial como ferramenta de aprendizagem ativa, utilizando o NotebookLM para compreender como a experiência do usuário (UX) pode ser otimizada em produtos digitais.

O estudo aborda conceitos como personalização, predictive UX, design systems e métricas de performance, simulando cenários reais de plataformas digitais, como sistemas de compra de ingressos (cinema, teatro e eventos).

Objetivos específicos:
- Utilizar IA para estudar e sintetizar conceitos de UX  
- Entender como IA pode melhorar a experiência do usuário  
- Explorar técnicas de personalização e previsão de comportamento  
- Aplicar pensamento crítico na análise de respostas da IA  
- Criar um miniguia de estudo reutilizável

## 📖 Miniguia de Estudo

### 🧾 Resumo Estruturado

A experiência do usuário (UX) em produtos digitais está diretamente ligada à capacidade do sistema de reduzir fricções e antecipar necessidades. Em plataformas de compra de ingressos, por exemplo, o usuário geralmente busca rapidez, clareza e recomendações relevantes — qualquer barreira no fluxo pode impactar diretamente a conversão.

Nesse contexto, a Inteligência Artificial atua como um motor de eficiência, permitindo que sistemas se adaptem dinamicamente ao comportamento do usuário.

Principais aplicações:

- **Personalização**: exibição de eventos com base no histórico de navegação e preferências do usuário  
- **Predictive UX**: antecipação de ações, como sugerir horários, locais ou tipos de evento com maior probabilidade de interesse  
- **Otimização do funil de compra**: redução de etapas e simplificação do checkout com base em padrões de comportamento  
- **Design Systems com IA**: padronização e escalabilidade da interface com apoio de automação  
- **Análise de performance**: uso de métricas como Core Web Vitals para identificar gargalos que impactam a experiência  

Em plataformas de ingressos, essas aplicações podem significar desde recomendar um filme no momento certo até reduzir o tempo necessário para finalizar uma compra — impactando diretamente métricas como conversão, retenção e satisfação do usuário.

---

### 📚 Glossário

- **UX (User Experience)**: experiência do usuário ao interagir com um produto  
- **Personalização**: adaptação da experiência com base em dados  
- **Predictive UX**: uso de IA para prever comportamento do usuário  
- **Machine Learning**: algoritmos que aprendem com dados  
- **Core Web Vitals**: métricas de performance que impactam UX  
- **Design System**: conjunto de padrões para construção de interfaces
